# Frontend Angular

Frontend tres simple pour la mini plateforme TODO.

## Pages

- `/login` : connexion
- `/users` : ajout et liste des utilisateurs
- `/todos` : gestion des taches

## Lancement local

```bash
npm install
npm start
```

Application disponible sur `http://localhost:4200`.

Toutes les requetes HTTP passent par l'API Gateway. L'URL de base est centralisee dans `src/environments/environment.ts`.
